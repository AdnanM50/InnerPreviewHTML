Admin Dashboard Overview

Admin Profile Settings

  • Basic Information: Name, Email, Phone Number
  • Change Password: Option to update password for security

Key Metrics Displayed

  • Total Users: Displays the total number of users
  • Total Trainers: Shows the total number of trainers
  • Total Groups: Indicates the number of groups created
  • Total Orders: Tracks the number of orders placed
  • Complete Orders: Shows the number of completed orders
  • Active Subscriptions: Displays total active subscriptions

User Management

  • Role-based Access Control
  • User Profile Management
  • Subscription Management
  • Password Management
  • Search and Filter: Find users by name,email or status

Trainer

  • Add Trainer: Register a new trainer
  • View Trainer: View details like name, phone, email
  • Delete Trainer: Remove a trainer permanently

Groups

  • Group Images: Upload images for each group
  • Group Facilities: Add or update facilities for each group
  • Assigned Trainer: Assign a trainer to a group
  • Group Status: Manage active or inactive status

Product Management

  • Category: Organize products for easy navigation
  • Products: Manage product details, price, SKU, etc.
  • Orders: Track order history and status
  • Coupon: Create and manage discount codes
  • Testimonial: Display customer feedback

Additional Features

  • Schedule: Manage availability and time slots
  • Service: List services with icons and descriptions
  • Events: Manage event details, location, images
  • Newsletter: Manage subscriber emails
  • Gallery: Store images and videos
  • Pricing Plan: Display pricing plans
  • Currency: Manage currency options
  • Payment Methods: Available payment options
