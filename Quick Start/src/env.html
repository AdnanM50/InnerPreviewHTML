Local Admin Configuration

Initial Local Setup

Open your local application by entering http://localhost:3000 in your browser. The setup page will appear if this is the first time configuration. Follow the steps below to complete the setup.

  • 1. Create Admin: Enter basic information such as username, email, phone number, and password to set up the initial admin account.
  • 2. Database Setup: Enter your local MongoDB connection string and specify the application or company name. If you need help generating a MongoDB connection string, refer to the MongoDB Setup Guide.

  • Ensure that your AWS S3 bucket information is correctly filled out. Click here for AWS S3 setup instructions

  • 3. Restart the Local Server: Open a terminal in the backend project folder, then restart your local server by running: 
    yarn start

Refresh your browser on http://localhost:3000 to view the homepage.
